关于node.js的npm命令无反应解决方法

John Doe 发布于 2016-04-10

npm完全无反应，不管是热启动(npm run dev)还是安装(npm install)

下标不停在哪里闪，但是没有反应……

解决方案: 删除.npmrc文件

​ 强调：不是安装node.js目录npm模块下的那个npmrc文件

​ 而是在c;\users\账户\下的.npmrc文件
